Control de Stock - EntregaFinal+Montealgre

Instrucciones: Programa para verificar si es necesario comprar o no comprar cantidades de un producto en relación a tu categoría y abastecer al stock máximo.

Paso 1: Ingresar el stock actual del producto a consultar (puede ser uno, varios o todos los productos).
Paso 2: Apretar el botón "Verificar Stock" para ver el análisis.
Paso 3: Aparecerán en rojo el texto del producto que se requiera generar compra.
Paso 4: Apretar botón "Limpiar" para borrar lo ingresado y comenzar nuevamente.

pd: El stockMax es un valor declarado para cada producto. Si es necesario comprar, indicará comprar una cantidad para llegar al stock máximo.
